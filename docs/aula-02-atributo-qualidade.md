## Problema Norteador do PBL — Projeto Local Eats

| Problema identificado | Atributo de qualidade afetado | Justificativa técnica |
|----------------------|------------------------------|----------------------|
| Lentidão em horários de pico | Atributo da Eficiência de Desempenho | Tempo de resposta elevado prejudica experiência do usuário. |
| Telas confusas e pouco intuitivas | Atributo da Usabilidade | Dificulta a navegação o que faz com que o usuário tenha dificuldade em usar o sistema. |
| Buscas com resultados incorretos | Atributo da Adequação Funcional | Resultados Incorretos nas buscas indicam que as funcionalidades não estão funcionando como deveriam. |
| Falhas em determinados modelos de smartphone | Atributo da Compatibilidade | O sistema não funciona em alguns dispositivos demonstra que não está adaptado para diferentes modelos ou sistemas. |
| Dificuldade para concluir ações simples | Atributo da Usabilidade | Usuário com dificuldade de realizar tarefas básicas demonstra que o sistema não é fácil de usar. |
| Avaliações desaparecem após atualização da página | Atributo da Confiabilidade | O sistema perde informações ou não mantém dados corretamente após atualização, o que prejudica a confiança do usuário na plataforma. |
| Inconsistências entre versão web e versão mobile | Atributo da Portabilidade | O sistema apresenta comportamentos diferentes entre as plataformas, o que indica falta de padronização entre as versões. |
